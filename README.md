# AutoCat - 乌龟服德鲁伊增强插件

## 简介

**AutoCat** 是专为魔兽世界乌龟服德鲁伊设计的全能插件，集成了猫德、熊德、治疗等多种形态的自动化功能，让您的德鲁伊体验更加流畅。

## 主要功能

### 🐱 猫德功能

- **一键攻击宏**：支持 4 种攻击模式（自动选择、背刺流、双流血、流血撕碎）
- **智能形态管理**：自动保持猎豹形态，支持猫变猫功能
- **技能自动化**：自动保持猛虎之怒、补充精灵之火、自动突袭
- **神像舞**：自动切换神像以最大化输出
- **目标保护**：避免攻击玩家目标，自动切换敌方目标
- **背后判断**：支持 UnitXP 模块强化正反面判断
- **自动保命**：OT 时先吃有限无敌，之后自动变熊保命

### 🐻 熊德功能

- **技能时机**：可配置槌击、野蛮撕咬、挥击等技能使用时机
- **生存技能**：自动狂暴回复、狂怒、狂暴等
- **仇恨管理**：可配置嘲讽、挫志咆哮等
- **技能通报**：可选择是否通报重要技能使用

### 🌿 治疗功能

- **毛治疗量**：在团队满血时利用 6T3 特效自动毛治疗量
- **愈合等级**：可自定义愈合术等级（1-9 级）

### 🎒 自动化功能

- **饰品管理**：自动使用上部/下部饰品，带可用性检测
- **消耗品**：自动使用治疗石、草药茶、献祭之油
- **自动拾取**：可设置拾取间隔，解放双手（本功能不开放）
- **能量优化**：智能变身回能，提升输出效率

### 🔧 调试功能

- **模块状态检测**：显示 SuperWoW、UnitXP 模组状态
- **调试模式**：可通过命令行控制调试开关和等级
- **详细日志**：帮助排查问题和优化设置

## 安装方法

1. 下载插件压缩包
2. 解压到魔兽世界目录下的 `Interface/AddOns/` 文件夹
3. 启动游戏，在插件列表中启用 "AutoCat"

## 使用方法

### 基本使用

1. 游戏中会看到加载成功提示
2. 创建宏，内容填入：`/script AutoCat()` 或直接使用 `/autocat` 命令
3. 点击小地图上的猫德图标可打开设置界面

### 参数化使用

- `/script AutoCat(1)` - 自动选择攻击模式
- `/script AutoCat(2)` - 仅背刺流
- `/script AutoCat(3)` - 仅双流血
- `/script AutoCat(4)` - 流血撕碎

### 熊德命令

- `/autobear` 或 `/AutoBear` - 执行熊德攻击
- `/script AC.Bear:PullAll()` - 熊德攻击（不嘲讽版本：`AC.Bear:PullAll(true)`）

### 治疗命令

- `/healrej` 或 `/rejuv` - 执行团队治疗
- `/script HealCat()` - 执行团队治疗
- `/healrej rank` - 查看当前愈合等级
- `/healrej rank <1-9>` - 设置愈合等级
- `/healrej help` - 显示治疗命令帮助

### 调试命令

- `/autocat debug on` - 开启调试模式
- `/autocat debug off` - 关闭调试模式
- `/autocat debug level 1` - 设置调试等级（1-3）

## 设置界面

点击小地图图标或输入 `/cat` 可打开设置界面，包含以下四大分类：

### 猫德设置

- 战斗模式选择
- UnitXP 背后判断
- 技能自动化开关
- 目标保护选项
- OT 变熊开关

### 熊德设置

- 各技能使用时机配置
- 生存技能阈值设置
- 技能通报选项

### 治疗设置

- 愈合术等级配置

### 共通设置

- 饰品使用开关
- 消耗品使用阈值
- 调试相关选项

## 推荐模块

虽然不是必需的，但建议安装以下模块以获得最佳体验：

- **SuperWoW**：增强战斗日志和事件检测
- **UnitXP_SP3**：强化目标选择和背后判断
- **Cursive**：更准确的 DOT 效果检测

## 注意事项

- 插件会自动检测并显示推荐模块的安装状态
- 某些功能在没有相关模块时会使用备用方案
- 建议根据自己的游戏习惯调整各项设置
- OT 变熊功能需要熊德技能才能正常工作

## 常见问题

**Q: 为什么攻击不生效？**
A: 请检查目标是否为玩家，插件默认会避免攻击玩家目标。

**Q: 如何恢复默认设置？**
A: 删除 `WTF/Account/<账号>/SavedVariables/` 下的相关文件后重启游戏。

**Q: 为什么某些功能显示"未导入"？**
A: 这表示相关的可选模块未安装，大部分功能仍可正常使用。
